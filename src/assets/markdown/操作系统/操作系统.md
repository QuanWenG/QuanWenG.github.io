1. 什么是操作系统
- 管理计算机硬件和软件资源的程序
- 本质上是个软件
- 屏蔽了硬件层的复杂性
- 内核是应用程序和硬件的桥梁

2. 操作系统的功能
- 进程和线程管理
- 存储管理
- 文件管理
- 设备管理
- 网络管理
- 安全管理

3. 常见操作系统
- windows
- unix
- linux
- Mac OS
- Harmony OS

4. 用户态和内核态
- 用户态 可以读取用户程序的数据，低权限
- 内核态 几乎可以访问任何资源

5. 为什么要有用户态和内核态
- 安全和稳定

6. 用户态和内和态的切换方式
- 系统调用 主动
- 中断 被动
- 异常 被动
- 状态切换是有性能开销的
